# Blazorwind

A Blazor WebAssembly App with Tailwind CSS. https://tailwindcss.com/

To get started:

1. Clone the repository:

    ```bash
    git clone https://github.com/eshanks/blazorwind.git blazorwind
    cd blazorwind
    ```

2. Install the dependencies:

    ```bash
    # Using npm
    npm install

    # Using Yarn
    yarn
    ```
    
3. Build tailwind:

    ```bash
    # Using npm
    npm run build-tailwind

    # Using Yarn
    yarn build-tailwind
    ```

Start building...
